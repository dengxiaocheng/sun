# sun 支线：邓孝程家乡旅途线

## 支线定位

本支线是《雾线下的译者》中一条偏正向发展的关系线。它不把“奔赴”写成单方面牺牲，而把一次极长、复杂、疲惫的旅途设计成一套可玩的压力管理关卡。

孙铭欣为了见邓孝程，从学校出发，经历：

1. 从学校打车去地铁站。
2. 坐一个半小时地铁到机场。
3. 在机场等一个小时登机。
4. 坐三个小时飞机到邓孝程所在省份的机场。
5. 再坐一个小时地铁到火车站。
6. 再坐一个半小时火车到邓孝程家附近。
7. 在小站或乡镇路口与邓孝程见面。

这不是一段“跳过即可”的转场，而是一整章玩法。玩家要管理路线、体力、电量、时间缓冲、焦虑和关系期待。

## 正向发展原则

这条支线的核心不是“孙铭欣跑很远，所以她更爱邓孝程”，而是：

> 孙铭欣愿意奔赴，但她也开始要求对等。邓孝程如果真的珍惜她，就必须把爱从理想、艺术、口号，落实到接站、确认、照顾和安排休息。

邓孝程的成长不再用宏大宣言体现，而用具体行为体现：查车次、确认接站、带水和食物、减少不必要社交、让她先休息。
