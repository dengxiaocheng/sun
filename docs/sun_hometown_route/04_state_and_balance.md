# 04｜状态与平衡

## 临时状态定义

| 状态 | 初始值 | 下降来源 | 上升来源 |
|---|---:|---|---|
| 旅程余裕 U | 60 | 堵车、迷路、反复确认、临时改路 | 路线单、提前确认、问路、合理查看路线 |
| 体力 F | 75 | 交通段消耗、背包过重、硬学、焦虑发作 | 吃饭、闭眼休息、飞机睡觉、候机休息 |
| 手机电量 C | 80 | 导航、刷手机、联系、查路线 | 充电宝、机场充电、减少无效刷屏 |
| 行李负担 L | 由打包决定 | 带过多复习资料、礼物、衣服 | 简化行李 |
| 连接感 K | 40 | 邓孝程不回、回应含糊、不接站 | 具体接站、查车次、提供休息安排 |

## 行李结算

行前打包界面最多选五件。每件有利弊。

| 物品 | 效果 |
|---|---|
| 充电宝 | C 下降速度减半。 |
| 保温杯 | F 低于 50 时可恢复一次。 |
| 耳机 | 地铁和候机时 A 增长降低。 |
| 考研单词小册 | 候机/地铁可轻复习，P 小幅增加。 |
| 纸质路线单 | C 低时仍可避免迷路。 |
| 眼罩 | 飞机休息效果增强。 |
| 小零食 | F 低时恢复一次。 |
| 给邓孝程家的小礼物 | T 潜在增加，L 增加。 |
| 备用外套 | 到达后如果天气冷，R 不下降。 |
| 过多复习资料 | P 潜在增加，但 L 大幅增加，A 增加。 |

| 交通段 | U | F | C | A |
|---|---:|---:|---:|---:|
| 打车 | -5 | -2 | -3 | +3 |
| 地铁去机场 | -5 | -10 | -8 | +8 |

| 飞机 | 0 | -15 | -2 | +5 |
| 目的机场找路 | -10 | -5 | -8 | +8 |
| 地铁去火车站 | -5 | -10 | -8 | +8 |
| 火车去家乡 | 0 | -8 | -3 | -5 |
| 小站见面 | 依据邓孝程表现 | 依据接站质量 | 0 | 依据接站质量 |

## 平衡原则

1. 玩家不应该因为复习而总是得到最优结果。旅途中硬学会损耗体力。
2. 玩家不应该因为恋爱而牺牲所有自我照顾。最优路线是“主动奔赴 + 清楚边界”。
3. 邓孝程的成长必须体现在具体行为，不体现在空泛表白。
4. 低分支不应绝望。即便邓孝程失职，孙铭欣也可以选择休整和自救。
