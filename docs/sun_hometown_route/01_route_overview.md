# 01｜路线总览：《去有风的县城》

## 一句话概念

孙铭欣从学校出发，完成一段几乎横跨一整天的复杂旅程，去邓孝程的家乡见他。旅途中的每一次等待、换乘、确认、疲惫和自我照顾，都是她把焦虑转化为行动的练习。

## 推荐放置位置

建议放在以下两个时间点之一：

1. **初试之后，成绩出来之前**：考研主压力暂时下降，适合进入关系修复与生活拓展。
2. **阶段性复习任务完成后**：作为关系线奖励章节，但必须避免破坏主线复习节奏。

不建议放在考前夜或关键模考前，因为那会让旅程显得像逃避。

## 旅程链条

| 顺序 | 场景 | 预计时长 | 玩法重点 |
|---|---|---:|---|
| T00 | 行前一夜，宿舍 | 前一天晚上 | 打包、路线确认、边界沟通。 |
| T01 | 学校门口打车 | 约 20-40 分钟 | 离开熟悉环境，首次焦虑翻译。 |
| T02 | 地铁去机场 | 1.5 小时 | 长时间移动、微行动选择、电量管理。 |
| T03 | 机场候机 | 1 小时 | 补给点：吃饭、充电、确认接站。 |
| T04 | 飞机 | 3 小时 | 离线内心关卡，不能靠消息确认关系。 |
| T05 | 目的省份机场 | 约 20-40 分钟 | 陌生环境、问路、导航、电量焦虑。 |
| T06 | 地铁去火车站 | 1 小时 | 第二轮疲惫，身体感上升。 |
| T07 | 火车去家乡附近 | 1.5 小时 | 从压迫交通转向田野、池塘、油菜花。 |
| T08 | 小站见面 | 抵达 | 关系落地：邓孝程是否具体承担。 |

总体验不是“七小时跳过”，而是玩家在七小时里不断选择：

- 要不要休息。
- 要不要反复确认。
- 要不要吃东西。
- 要不要向邓孝程提出具体需求。
- 要不要把焦虑翻译成下一段路。

## 情绪曲线

| 阶段 | 情绪 |
|---|---|
| 行前 | 不确定、羞耻感、期待和担心混杂。 |
| 前半程 | 冷灰、拥挤、时间压力。 |
| 飞机段 | 离线、内向、不可确认关系。 |
| 目的地机场到火车站 | 疲惫加重，行动感仍在。 |
| 火车窗外 | 视觉逐渐转暖，田野和电线出现。 |
| 抵达 | 关系被现实检验。 |
| 第二天家乡散步 | 如果邓孝程回应良好，进入低饱和暖光、油菜花、池塘、乡路。 |
