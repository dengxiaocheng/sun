# 03｜完整节点流程

## T00 行前一夜：宿舍

孙铭欣看着路线图，意识到见邓孝程不是一句“我去找你”就能解决的事，而是一整天的交通链条。

### 焦虑翻译

原始焦虑句：

> 我跑这么远，是不是显得很廉价？

| 选项 | 结果 |
|---|---|
| “我是在倒贴，他应该主动来见我。” | A +15，B -5 |
| “我可以主动奔赴，但我不该独自承担全部成本。” | B +15，解锁“确认接站和休息安排”。 |
| “这趟路很难，所以我要先把它拆成一段一段。” | U +10，M +5 |

### 行动选择

| 行动 | 结果 |
|---|---|
| 熬夜反复查路线 | U +5，R -15，A +10 |
| 写一张简洁路线单 | U +15，A -5 |
| 给邓孝程发“你能不能到站接我” | B +10，K 依据回应变化 |
| 什么都不准备，明天再说 | U -20，A +10 |

## T01 学校门口打车去地铁站

司机问她是不是去机场。孙铭欣忽然觉得自己像在做一件夸张的事。

| 想法 | 结果 |
|---|---|
| “我太夸张了，为了见他跑这么远。” | A +10 |
| “我正在做一件有成本的决定，所以我也有权利要求被认真对待。” | B +10，M +5 |
| “先到地铁站，不评价整趟人生。” | A -5，U +5 |

| 行动 | 结果 |
|---|---|
| 一直看导航 | A +5，U +5 |
| 只确认一次路线后闭眼休息 | A -5，F +5 |
| 给邓孝程发“我出发了” | K 依据回应变化 |
| 刷恋爱帖子 | A +15，T -5 |

## T02 地铁 1.5 小时去机场

这一段每 30 分钟出现一次行动选择，共三次。

| 行动 | 效果 |
|---|---|
| 背 15 个单词 | P +3，F -5 |
| 听白噪音闭眼 | A -8，F +5 |
| 看窗外 | A -5，M +3 |
| 反复看邓孝程有没有回消息 | A +10，C -8 |
| 给自己写一句焦虑翻译 | A -10，M +8 |
| 查看下一段路线一次 | U +5，C -3 |

如果邓孝程迟迟不回，触发：

> 我都在路上了，他还不回，是不是根本不在乎？

| 翻译 | 结果 |
|---|---|
| “他不回就是不爱我。” | A +15 |
| “不回消息让我难受，但我可以到机场后再确认具体安排。” | A -5，B +5 |
| “我需要的是接下来怎么被接住，而不是现在猜他心里有没有我。” | A -10，B +10，解锁机场通话选项 |

## T03 机场候机一小时

机场是补给点，也是关系确认点。

| 互动点 | 效果 |
|---|---|
| 充电座 | C +30 |
| 便利店 | F +15，花费增加 |
| 候机椅 | F +10，A -5 |
| 登机口屏幕 | U +10，反复确认会 A +5 |
| 安静角落 | 呼吸练习，A -15 |
| 发消息给邓孝程 | K/T/B 变化 |

关键消息：

> 我落地之后还要坐地铁去火车站，再坐火车。你能不能确认一下，我到小站后你在哪里接我？我可能会很累。

邓孝程回应分三档：

| 回应 | 条件 | 效果 |
|---|---|---|
| “我到时候看情况吧。” | T 低、B 低 | K -15，A +10 |
| “你到了给我说。” | 中性 | K 不变 |
| “我查过了，你那班车到 19:42，我在出站口等你。先带你吃饭，不急着见别人。” | T 高或 B 高 | K +25，T +15，A -15 |

## T04 飞机三个小时

飞机上不能联网，玩家不能通过消息确认关系，只能处理身体和想法。

| 行动 | 效果 |
|---|---|
| 睡一会儿 | F +20，A -5 |
| 看考研资料 | P +5，F -10 |
| 写一封不一定发送的信 | B +10，M +10 |
| 一直想象最坏结果 | A +20 |
| 看云层 | A -8，M +5 |

焦虑翻译：

> 如果他见到我之后还是冷淡，我这一天就白费了。

| 翻译 | 结果 |
|---|---|
| “那我就证明自己真的没人要。” | A +20 |
| “这趟路不是只为了证明他爱不爱我，也是为了看清这段关系能不能落地。” | B +15，M +10 |
| “我可以期待他，但不能把自己的价值托运给他。” | A -10，B +10 |

## T05 目的省份机场

她下飞机后发现机场很大，地铁口不好找，手机电量只剩 32%。

| 行动 | 效果 |
|---|---|
| 立刻给邓孝程打电话 | K 变化，C -8 |
| 先找到地铁口再联系 | U +5，C -3 |
| 原地慌十分钟 | U -10，A +10 |
| 找工作人员问路 | U +10，M +5，A -5 |

## T06 地铁一小时到火车站

第二段地铁更疲惫。背包肩带勒疼，站名陌生，火车还有 40 分钟检票。

| 行动 | 效果 |
|---|---|
| 靠边站稳，闭眼听报站 | F +5，A -5 |
| 继续背单词 | P +3，F -8 |
| 给邓孝程发“我现在很累，但还在路上” | B +5，K 变化 |
| 刷短视频麻痹自己 | A -3，C -12 |
| 查看火车站换乘路线 | U +10，C -5 |

## T07 火车一个半小时到家乡附近

火车开出城市。窗外开始出现田、树、低矮房子、电线、池塘和油菜花。画面从冷灰蓝转向暖黄和浅绿。

| 行动 | 效果 |
|---|---|
| 睡一会儿 | F +15 |
| 看窗外 | A -15，M +10 |
| 写下“我已经走到这里了” | M +15 |
| 给邓孝程发窗外照片 | T +5，K +5 |
| 担心见他家人 | A +10 |

焦虑翻译：

> 我马上要到他的地方了，我会不会格格不入？

| 翻译 | 结果 |
|---|---|
| “我肯定会出丑。” | A +15 |
| “陌生不是危险，只是我还没认识这里。” | A -10，M +5 |
| “我可以慢慢看，慢慢问，不需要立刻表现得合适。” | A -10，R +5 |

## T08 小站见面

邓孝程是否接住她，决定支线走向。

### 低质量接站

邓孝程迟到，或者只说：“你怎么这么累。”

结果：A +20，T -10。

玩家可选择：

- 忍着说没事。
- 直接表达“我今天真的很累”。
- 要求先休息，不继续社交。
- 暂时订酒店/民宿自己休整。

### 中等接站

邓孝程在出站口等，但不太会表达。他帮她拿包，说：“先去吃点东西吧。”

结果：T +10，K +10。

### 高质量接站

邓孝程提前到站，带了水和小面包。

> 今天不安排别的了。你坐了这么久车，先吃饭，回去睡觉。明天我再带你看我家那边。

结果：T +25，K +25，B +10，A -20。
